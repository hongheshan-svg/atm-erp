## 图标设计建议

- 未选中状态：灰色 (#999999)
- 选中状态：主题蓝色 (#1890ff)
- 格式：PNG，透明背景
- 尺寸：81x81 像素（3倍图）

可以使用 [Iconfont](https://www.iconfont.cn/) 或 [IconPark](https://iconpark.oceanengine.com/) 获取图标资源。
