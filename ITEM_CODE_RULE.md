# 物料编码规则说明

## 编码规则

物料编码采用 **10位数字** 组成，格式如下：

```
一级代码(1位) + 二级代码(1位) + 年份(2位) + 三级流水号(6位)
```

### 一级代码（1位）

| 代码 | 含义 |
|------|------|
| 1    | 有图 |
| 2    | 无图 |

### 二级代码（1位）

| 代码 | 含义 |
|------|----------|
| 1    | 机加 |
| 2    | 钣金 |
| 3    | 特殊工艺 |
| 4    | 其他 |
| 5    | 机械类 |
| 6    | 电气类 |
| 7    | 耗材辅料 |
| 8    | 办公用品 |

### 年份（2位）

- **一级代码为1（有图）**：登记为当前年份后两位
  - 例如：2025年 → 25
- **一级代码为2（无图）**：年份登记为 **99**

### 三级流水号（6位）

- 根据年份为一个循环累加
- 同一个年份不允许出现相同的三级流水号
- 格式：000001 ~ 999999
- 不足6位时前面补0

## 编码示例

### 示例1：有图 + 机加
```
一级代码: 1 (有图)
二级代码: 1 (机加)
年份: 25 (2025年)
流水号: 000001 (第一个)

完整编码: 1125000001
```

### 示例2：无图 + 办公用品
```
一级代码: 2 (无图)
二级代码: 8 (办公用品)
年份: 99 (固定)
流水号: 000001 (第一个)

完整编码: 2899000001
```

### 示例3：有图 + 钣金
```
一级代码: 1 (有图)
二级代码: 2 (钣金)
年份: 25 (2025年)
流水号: 000123 (第123个)

完整编码: 1225000123
```

## 使用方法

### 在物料管理页面生成编码

1. **进入物料管理**
   - 导航至：基础数据 > 物料管理
   - 点击"添加物料"按钮

2. **打开编码生成器**
   - 在"物料编码"输入框右侧
   - 点击"生成"按钮

3. **选择编码参数**
   - **一级代码**：选择"有图"或"无图"
   - **二级代码**：从下拉列表选择物料类别
     - 1-机加
     - 2-钣金
     - 3-特殊工艺
     - 4-其他
     - 5-机械类
     - 6-电气类
     - 7-耗材辅料
     - 8-办公用品

4. **生成并应用**
   - 点击"生成编码"按钮
   - 查看预览的编码
   - 点击"应用"按钮将编码填入表单

5. **完成物料信息**
   - 填写其他必填字段（物料名称、规格等）
   - 点击"确定"保存

### 手动输入编码

如果不使用自动生成，也可以手动输入编码，但必须符合以下规则：

- **长度**：必须是10位数字
- **格式**：一级代码(1位) + 二级代码(1位) + 年份(2位) + 流水号(6位)
- **唯一性**：系统会检查编码是否已存在

## 编码规则的优势

1. **结构化**：编码包含物料的关键分类信息
2. **可追溯**：通过年份可以追溯物料创建时间
3. **易识别**：通过一级/二级代码快速识别物料类型
4. **自动化**：流水号自动累加，避免重复
5. **扩展性**：支持999,999个物料编码（每个前缀）

## 技术实现

### 后端实现

- **编码生成器**：`ItemCodeGenerator` 类
- **线程安全**：使用数据库行锁确保并发安全
- **流水号管理**：基于前缀（一级+二级+年份）自动累加
- **API接口**：
  - `POST /api/masterdata/items/generate_code/` - 生成编码
  - `GET /api/masterdata/items/get_level2_choices/` - 获取二级代码选项

### 前端实现

- **编码生成对话框**：可视化选择编码参数
- **实时预览**：生成后立即显示编码
- **一键应用**：点击按钮将编码填入表单

## 常见问题

### Q1: 流水号用完了怎么办？
A: 每个前缀（一级+二级+年份）支持999,999个编码。如果用完，需要更换年份或二级代码。

### Q2: 可以修改已有物料的编码吗？
A: 建议不要修改已使用的物料编码，因为可能已被其他单据引用。如需修改，请确保没有关联数据。

### Q3: 无图物料的年份为什么是99？
A: 根据业务规则，无图物料不按年份区分，统一使用99作为年份标识。

### Q4: 可以跳号吗？
A: 流水号是连续的，不支持跳号。如果删除了某个物料，其编码不会被重用。

### Q5: 不同年份的编码会重复吗？
A: 不会。因为年份是编码的一部分，不同年份的编码前缀不同，所以不会重复。

## 注意事项

1. **编码唯一性**：每个物料编码在系统中必须唯一
2. **不可随意修改**：物料编码一旦创建，建议不要修改
3. **规范使用**：建议使用自动生成功能，避免手动输入错误
4. **分类准确**：选择二级代码时要准确，因为编码会永久包含这个分类信息
5. **年份管理**：每年1月1日后，有图物料的编码年份会自动更新
