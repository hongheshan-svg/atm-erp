## CRM客户管理

### 1. 销售线索管理

**流程图:**
```
获取线索 → 录入系统 → 初步筛选 → 跟进沟通 → 转化商机/放弃
```

**操作步骤:**
1. 【销售人员】进入「CRM客户管理 > 销售线索」
2. 点击「新增线索」填写客户信息
3. 定期跟进，更新线索状态
4. 确认有价值后，点击「转为商机」

**状态流转:**
```
新建 → 已联系 → 有意向 → 已转化/已放弃
```

---

### 2. 销售商机管理

**流程图:**
```
商机创建 → 需求确认 → 方案设计 → 报价谈判 → 合同签订/丢单
```

**操作步骤:**
1. 【销售人员】进入「CRM客户管理 > 销售商机」
2. 从线索转化或直接新建商机
3. 关联客户、填写预期金额和成交概率
4. 推进各阶段，记录跟进情况
5. 成交后转为销售合同

**阶段说明:**
| 阶段 | 概率 | 说明 |
|------|------|------|
| 初步接触 | 10% | 刚建立联系 |
| 需求确认 | 30% | 已确认客户需求 |
| 方案/报价 | 50% | 已提交方案 |
| 商务谈判 | 70% | 进入价格谈判 |
| 赢单 | 100% | 签订合同 |
| 丢单 | 0% | 项目失败 |

---

### 3. 客户档案管理

**操作步骤:**
1. 【销售/客服】进入「CRM客户管理 > 客户档案」
2. 查看/编辑客户基本信息
3. 关联联系人、项目、订单等信息
4. 记录客户拜访和沟通记录

---

### 4. 销售报价流程

**流程图:**
```
需求分析 → BOM估算 → 生成报价单 → 客户确认 → 转为合同
```

**操作步骤:**
1. 【销售人员】进入「CRM客户管理 > 销售报价」
2. 点击「新增报价」，选择客户和商机
3. 添加报价明细（产品/服务/工时）
4. 提交审批（如需要）
5. 客户确认后，点击「转为合同」

---

### 5. 销售合同管理

**流程图:**
```
创建合同 → 法务审核 → 总经理审批 → 签订生效 → 执行交付 → 回款结算
```

**状态流转:**
```
草稿 → 待审核 → 已审批 → 执行中 → 已完成/已终止
```

## PLM产品研发

### 1. 需求管理

**适用人员:** 销售、项目经理、技术人员

**流程图:**
```
客户需求收集 → 需求录入 → 评审分析 → 技术可行性 → 需求确认 → 立项
```

**操作步骤:**
1. 【销售/项目经理】进入「PLM产品研发 > 需求管理」
2. 点击「新增需求」，填写需求信息
3. 上传客户提供的技术文件作为附件
4. 关联客户、项目、商机
5. 分配负责人进行评审
6. 更新需求状态和评审结论

**需求分类:**
| 类型 | 说明 |
|------|------|
| 功能需求 | 产品/设备功能要求 |
| 性能需求 | 速度、精度、效率等 |
| 界面需求 | 操作界面、人机交互 |
| 安全需求 | 安全防护要求 |
| 其他需求 | 包装、运输、培训等 |

---

### 2. 方案设计

**流程图:**
```
需求分析 → 概念设计 → 初步方案 → 方案评审 → 详细设计 → 方案确认
```

**操作步骤:**
1. 【技术人员】进入「PLM产品研发 > 方案设计」
2. 基于需求创建方案
3. 编写技术方案文档
4. 提交评审，记录评审意见
5. 修改完善后确认方案

---

### 3. 产品配置器

**用途:** 根据客户需求快速配置产品参数和选型

**操作步骤:**
1. 【技术/销售】进入「PLM产品研发 > 产品配置器」
2. 选择产品模板
3. 配置各参数选项
4. 生成配置清单和报价

---

### 4. BOM管理

**流程图:**
```
设计BOM → 工程BOM → 采购BOM → 生产BOM
```

**BOM类型:**
| 类型 | 用途 | 维护部门 |
|------|------|---------|
| 设计BOM | 产品结构定义 | 研发 |
| 工程BOM | 工艺路线关联 | 工艺 |
| 采购BOM | 采购清单 | 采购 |
| 生产BOM | 生产领料 | 生产 |

---

### 5. 图纸管理

**操作步骤:**
1. 【工程师】进入「PLM产品研发 > 图纸管理」
2. 上传图纸文件（支持PDF、DWG、STEP等）
3. 关联项目/产品/BOM
4. 设置版本号和变更记录
5. 提交审批（如需要）

**版本控制:**
```
V1.0 → V1.1 (小修改) → V2.0 (大改版) → ...
```

---

### 6. ECN工程变更

**流程图:**
```
变更申请 → 影响分析 → 评审决策 → 变更实施 → 验证确认 → 关闭
```

**操作步骤:**
1. 【工程师】进入「PLM产品研发 > ECN变更」
2. 创建变更申请，说明变更原因和内容
3. 选择影响的物料/BOM/图纸
4. 提交评审，相关部门确认
5. 批准后实施变更
6. 验证无误后关闭

## 项目管理

### 1. 项目立项流程

**流程图:**
```
项目申请 → 可行性分析 → 立项审批 → 项目启动 → 项目执行 → 项目验收 → 项目结项
```

**操作步骤:**
1. 【项目经理】进入「项目管理 > 项目列表」
2. 点击「新增项目」，填写项目信息
3. 关联客户、合同
4. 提交立项审批
5. 审批通过后启动项目

**项目状态:**
```
规划中 → 进行中 → 暂停 → 已完成/已取消
```

---

### 2. 任务管理 (WBS)

**流程图:**
```
任务分解 → 任务分配 → 任务执行 → 进度更新 → 任务完成
```

**操作步骤:**
1. 【项目经理】进入「项目管理 > 任务管理」
2. 选择项目，创建任务树（WBS）
3. 设置任务负责人、计划工期、开始/结束日期
4. 【执行人员】更新任务进度
5. 填报工时记录工作量

**任务状态:**
```
待办 → 进行中 → 已完成/已取消
```

---

### 3. 里程碑管理

**用途:** 标记项目关键节点，跟踪交付进度

**里程碑类型:**
| 类型 | 说明 |
|------|------|
| 设计里程碑 | 设计评审、图纸发布 |
| 采购里程碑 | 长周期物料到货 |
| 生产里程碑 | 组装完成、调试完成 |
| 交付里程碑 | 出厂验收、发货、现场验收 |
| 付款里程碑 | 关联回款节点 |

---

### 4. 工时填报

**流程图:**
```
选择日期 → 选择项目/任务 → 填写工时和内容 → 提交 → 审批 → 汇总到任务
```

**操作步骤:**
1. 【员工】进入「项目管理 > 工时填报」
2. 点击「填报工时」
3. 选择日期、项目、任务
4. 填写工作小时数和工作内容
5. 提交等待审批

**注意:** 
- 工时审批后自动汇总到对应任务的「实际工时」
- 建议每日及时填报，避免遗忘

---

### 5. 甘特图

**用途:** 可视化展示项目进度和任务依赖关系

**操作:**
1. 进入「项目管理 > 甘特图」
2. 选择项目查看时间轴
3. 拖拽调整任务时间
4. 查看关键路径和资源冲突

---

### 6. Bug跟踪

**流程图:**
```
发现问题 → 创建Bug → 分配处理 → 修复验证 → 关闭
```

**严重程度:**
| 级别 | 说明 |
|------|------|
| 致命 | 系统崩溃、数据丢失 |
| 严重 | 主要功能无法使用 |
| 一般 | 功能异常但可绕过 |
| 轻微 | 界面/体验问题 |

## 采购管理

### 1. 采购申请流程

**流程图:**
```
需求提出 → 创建申请 → 部门审批 → 采购执行
```

**操作步骤:**
1. 【需求人员】进入「采购管理 > 采购申请」
2. 点击「新增申请」，添加物料和数量
3. 填写需求日期和用途说明
4. 提交审批

---

### 2. 询比价流程

**流程图:**
```
采购申请 → 发起询价 → 供应商报价 → 比价分析 → 确定供应商
```

**操作步骤:**
1. 【采购员】进入「采购管理 > 询价管理」
2. 基于采购申请创建询价单
3. 发送给多个供应商
4. 收集报价，进行比价
5. 选择最优供应商

---

### 3. 采购订单流程

**流程图:**
```
创建订单 → 审批 → 发送供应商 → 供应商确认 → 跟踪交货 → 收货入库
```

**操作步骤:**
1. 【采购员】进入「采购管理 > 采购订单」
2. 点击「新增订单」，选择供应商
3. 添加订单明细
4. 提交审批
5. 审批后发送给供应商
6. 跟踪交货进度

**订单状态:**
```
草稿 → 已审批 → 已下单 → 部分收货 → 已完成/已取消
```

---

### 4. 收货入库流程

**流程图:**
```
供应商发货 → 到货通知 → 质量检验 → 入库确认 → 库存更新
```

**操作步骤:**
1. 【仓库人员】进入「采购管理 > 收货管理」
2. 选择采购订单，创建收货单
3. 核对数量和规格
4. 检验合格后确认入库
5. 系统自动更新库存

---

```
销售订单 → 主生产计划 → 物料需求计划 → 产能规划 → 生产排程
```

**操作步骤:**
1. 【生产计划员】进入「生产管理 > 生产计划」

**流程图:**
```
工序开始 → 领料 → 生产加工 → 工序报工 → 质检 → 工序完成
```

**操作步骤:**

3. 确认物料齐套后开始生产
4. 完成后提交报工

### 3. 调试记录

**用途:** 记录设备调试过程和问题

**操作步骤:**

---

### 4. 质量检验流程

**流程图:**
```

```

**操作步骤:**

2. 接收待检物料/产品
3. 按检验标准进行检验
4. 录入检验数据
5. 判定合格/不合格
6. 不合格品走让步/返工/报废流程

## 库存管理

- 进入「库存管理 > 库存查询」
- 按仓库/物料/批次查询库存

---

### 2. 库存调整/盘点

**流程图:**
```
创建盘点单 → 实盘录入 → 差异分析 → 审批调整 → 库存更新
```

3. 录入实际盘点数量

5. 差异超限需审批
6. 确认后更新库存

### 3. 库存移动
