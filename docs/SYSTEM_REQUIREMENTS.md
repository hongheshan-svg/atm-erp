## 一、文档概述

### 1.1 项目背景

本系统是针对**非标自动化行业**开发的一体化企业管理平台，整合了：
- **CRM** (Customer Relationship Management) - 客户关系管理
- **PLM** (Product Lifecycle Management) - 产品生命周期管理
- **ERP** (Enterprise Resource Planning) - 企业资源计划
- **MES** (Manufacturing Execution System) - 制造执行系统
- **OA** (Office Automation) - 协同办公

### 1.2 非标自动化行业特点

| 特点 | 说明 | 系统应对 |
|------|------|---------|
| 项目制生产 | 每个订单都是独特的非标设备 | 以项目为核心串联全流程 |
| 定制化设计 | 需要根据客户需求进行方案设计 | PLM模块支持需求-设计追溯 |
| 多品种小批量 | 物料种类繁多，批量小 | 灵活的BOM管理和MRP计划 |
| 现场调试 | 设备需要到客户现场安装调试 | 设备台账和现场服务管理 |
| 长周期交付 | 从设计到交付周期较长 | 里程碑管理和项目进度预警 |

### 1.3 系统架构

```

│                           非标自动化一体化平台                                │
├─────────────────────────────────────────────────────────────────────────────┤

│  │   CRM   │─▶│   PLM   │─▶│   ERP   │─▶│   MES   │─▶│   OA    │           │
│  │ 客户关系 │  │ 产品研发 │  │ 资源计划 │  │ 制造执行 │  │ 协同办公 │           │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘  └─────────┘           │
│       │            │            │            │            │                │
```
