## 1. 系统概述

### 1.1 系统简介

本ERP系统是一套面向制造企业的综合管理平台，集成了CRM客户管理、PLM产品研发、ERP资源管理、MES生产执行、OA协同办公等核心功能模块，帮助企业实现从销售、研发、采购、生产到财务的全流程数字化管理。

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    CRM 客户关系管理                       │
│  销售线索 → 销售商机 → 客户档案 → 报价 → 合同 → 售后    │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                    PLM 产品研发管理                       │
│  需求管理 → 方案设计 → 项目管理 → BOM管理 → 图纸管理    │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                    ERP 企业资源管理                       │
│  销售订单 → 采购管理 → 库存管理 → 财务管理              │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                    MES 生产执行系统                       │
│  生产计划 → APS排程 → 工单派工 → 质量检验 → 设备管理    │
└─────────────────────────────────────────────────────────┘
```

### 1.3 浏览器要求

- **推荐浏览器**：Chrome 90+、Firefox 88+、Edge 90+
- **屏幕分辨率**：建议 1920×1080 或以上

## 2. 登录与个人中心

### 2.1 系统登录

1. 打开浏览器，输入系统地址
2. 在登录页面输入**用户名**和**密码**
3. 点击「登录」按钮进入系统

> **默认管理员账号**：admin / admin123（首次登录后请立即修改密码）

### 2.2 个人中心

点击右上角头像，进入个人中心：

| 功能 | 说明 |
|------|------|
| 个人资料 | 查看/修改姓名、邮箱、手机、头像等 |
| 修改密码 | 修改登录密码（需输入原密码） |
| 通知设置 | 设置接收通知的方式（站内信、邮件等） |

### 2.3 退出登录

点击右上角头像 → 退出登录

## 3. 工作台（Dashboard）

### 3.1 功能概述

工作台是登录后的首页，提供：

- **待办事项**：显示待审批、待处理的任务
- **快捷入口**：常用功能的快速访问
- **数据概览**：关键业务指标的可视化展示
- **公告通知**：系统公告和重要通知

### 3.2 自定义仪表盘

管理员可通过「系统管理 → 仪表盘配置」自定义：

- 显示/隐藏各类图表
- 调整图表布局
- 配置数据刷新频率

## 4. CRM客户管理

### 4.1 销售线索

**路径**：CRM客户管理 → 销售线索

销售线索是潜在客户的初始信息录入。

| 操作 | 说明 |
|------|------|
| 新建线索 | 录入潜在客户的基本信息（姓名、公司、联系方式等） |
| 转为商机 | 将有意向的线索转化为销售商机 |
| 分配 | 将线索分配给指定销售人员跟进 |

### 4.2 销售商机

**路径**：CRM客户管理 → 销售商机

管理销售机会的跟进过程。

**商机阶段**：
1. 初步接触
2. 需求确认
3. 方案报价
4. 商务谈判
5. 签约成交
6. 丢单

### 4.3 客户档案

**路径**：CRM客户管理 → 客户档案

管理正式客户的详细信息。

**主要功能**：
- 客户基本信息（公司名称、地址、行业、规模）
- 联系人管理（多联系人）
- 银行账户信息
- 关联项目、订单、合同查看
- 批量导入/导出

**导入模板**：系统提供标准Excel模板，支持批量导入客户数据

### 4.4 客户跟进

**路径**：CRM客户管理 → 客户跟进

记录与客户的沟通历史。

| 字段 | 说明 |
|------|------|
| 跟进方式 | 电话、拜访、邮件、微信等 |
| 跟进内容 | 沟通的具体内容 |
| 下次跟进计划 | 计划的下次跟进时间和内容 |

### 4.5 信用管理

**路径**：CRM客户管理 → 信用管理

管理客户的信用额度和账期。

- 设置客户信用额度上限
- 监控客户当前欠款
- 信用预警提醒

### 4.6 销售报价

**路径**：CRM客户管理 → 销售报价

创建和管理销售报价单。

**操作流程**：
1. 点击「新建报价」
2. 选择客户
3. 添加产品明细（产品、数量、单价、折扣）
4. 设置有效期
5. 保存/提交审批
6. 审批通过后可「转为订单」

### 4.7 销售合同

**路径**：CRM客户管理 → 销售合同

管理销售合同的签订和执行。

**合同状态**：草稿 → 审批中 → 已生效 → 执行中 → 已完成

### 4.8 售后服务

**路径**：CRM客户管理 → 售后服务

管理售后工单和服务请求。

**工单类型**：
- 技术支持
- 维修服务
- 退换货
- 投诉处理

## 5. PLM产品研发

### 5.1 需求管理

**路径**：PLM产品研发 → 需求管理

收集和管理产品需求。

- 需求录入
- 需求评审
- 需求跟踪

### 5.2 方案设计

**路径**：PLM产品研发 → 方案设计

管理技术方案的设计过程。

### 5.3 产品配置器

**路径**：PLM产品研发 → 产品配置器

配置可选产品的选项和参数。

### 5.4 项目列表

**路径**：PLM产品研发 → 项目列表

管理所有项目的生命周期。

**新建项目**：
1. 点击「新建项目」
2. 填写项目信息：
   - 项目名称
   - 项目编号（可自动生成）
   - 客户
   - **关联销售订单**（可选，系统会自动带出订单信息）
   - 预算金额
   - 计划开始/结束日期
3. 保存

**关联销售订单说明**：
- 选择销售订单后，系统会显示订单详情（产品、金额、交期等）
- 自动填充客户、预算金额、结束日期
- 已关联项目的订单不会出现在可选列表中
- 项目取消后，关联的订单会被释放，可重新关联

**项目状态**：
- 规划中
- 进行中
- 已暂停
- 已完成
- 已取消

### 5.5 任务管理

**路径**：PLM产品研发 → 任务管理

管理项目下的具体任务。

**任务字段**：
- 任务名称
- 所属项目
- 负责人
- 计划开始/结束时间
- 优先级（高/中/低）
- 任务状态（待开始/进行中/已完成/已取消）
- 进度百分比

### 5.6 里程碑

**路径**：PLM产品研发 → 里程碑

设置和跟踪项目的关键节点。

### 5.7 甘特图

**路径**：PLM产品研发 → 甘特图

以甘特图形式展示项目进度。

- 支持拖拽调整任务时间
- 支持任务依赖关系显示
- 支持按项目/人员筛选

### 5.8 工时填报

**路径**：PLM产品研发 → 工时填报

员工填报工作时间。

| 字段 | 说明 |
|------|------|
| 项目 | 选择所属项目 |
| 任务 | 选择具体任务 |
| 日期 | 工作日期 |
| 工时 | 工作小时数 |
| 备注 | 工作内容描述 |

### 5.9 项目预警

**路径**：PLM产品研发 → 项目预警

监控项目风险和异常。

**预警类型**：
- 进度延期预警
- 预算超支预警
- 任务逾期预警

### 5.10 BOM管理

**路径**：PLM产品研发 → BOM管理

管理项目的物料清单（Bill of Materials）。

**主要功能**：
- 创建/编辑BOM条目
- 导入/导出Excel
- 按项目筛选
- **采购状态跟踪**

**BOM采购状态**：

| 状态 | 说明 |
|------|------|
| 未下单 | 尚未生成采购申请 |
| 申请中 | 已生成采购申请，待审批 |
| 已批准 | 采购申请已审批通过 |
| 已下单 | 已生成采购订单并确认 |
| 在途中 | 供应商已发货 |
| 已收货 | 已完成到货验收 |
| 部分收货 | 部分数量已收货 |
| 已退货 | 物料已退回供应商 |
| 已取消 | 采购已取消 |

**导出询价BOM**：
1. 使用筛选条件（有图/无图、物料类型、版本/品牌）
2. 选择需要的物料
3. 点击「导出筛选结果(询价用)」或「导出选中(询价用)」
4. 将导出的Excel发送给供应商询价

### 5.11 图纸管理

**路径**：PLM产品研发 → 图纸管理

管理项目相关的技术图纸。

- 上传/下载图纸文件
- 版本控制
- 按项目/物料关联

### 5.12 3D模型预览

**路径**：PLM产品研发 → 3D模型预览

在线预览3D模型文件。

### 5.13 ECN变更

**路径**：PLM产品研发 → ECN变更

管理工程变更通知（Engineering Change Notice）。

**ECN流程**：
1. 提出变更申请
2. 技术评审
3. 影响分析
4. 审批
5. 执行变更
6. 验证关闭

### 5.14 Bug跟踪

**路径**：PLM产品研发 → Bug跟踪

跟踪和管理产品缺陷。

**Bug状态**：新建 → 已确认 → 处理中 → 待验证 → 已关闭

### 5.15 知识库

**子模块**：
- **知识文章**：技术文档、操作规范
- **技术问题**：常见问题及解决方案
- **标准部件库**：可复用的标准组件
- **项目归档**：已完成项目的归档资料

## 6. ERP销售执行

### 6.1 销售订单

**路径**：ERP销售执行 → 销售订单

管理正式销售订单。

**新建销售订单**：
1. 点击「新建订单」或从报价单转换
2. 填写订单信息：
   - 客户（必填，需匹配系统中的客户）
   - 客户订单号
   - 交货日期
   - 付款条款
3. 添加产品明细
4. 保存/提交审批

**批量导入**：
1. 下载导入模板
2. 按模板格式填写数据
3. 主表填写订单信息，明细表填写产品
4. 明细表需填写「订单号」以关联主表
5. 上传导入

**订单状态**：草稿 → 已确认 → 执行中 → 已发货 → 已完成

### 6.2 发货管理

**路径**：ERP销售执行 → 发货管理

管理销售订单的发货。

**发货流程**：
1. 从销售订单生成发货单
2. 选择发货数量
3. 填写物流信息
4. 确认发货

### 6.3 销售业绩

**路径**：ERP销售执行 → 销售业绩

查看销售人员的业绩统计。

### 6.4 应收账款

**路径**：ERP销售执行 → 应收账款

管理客户应收款项。

- 查看应收明细
- 账龄分析
- 催款管理

### 6.5 回款计划

**路径**：ERP销售执行 → 回款计划

制定和跟踪回款计划。

## 7. ERP采购管理

### 7.1 MRP需求计划

**路径**：ERP采购管理 → MRP需求计划

物料需求计划（Material Requirements Planning）。

- 根据BOM和库存自动计算物料需求
- 生成采购建议

### 7.2 采购申请

**路径**：ERP采购管理 → 采购申请

管理采购申请的全流程。

**页面布局**：
1. **物料需求清单**（上方）
   - 显示所有项目中待采购的物料（状态为"未下单"）
   - 支持筛选：有图/无图、物料类型、版本/品牌
   - 支持导出询价Excel
   
2. **采购申请列表**（下方）
   - 已创建的采购申请记录

**创建采购申请**：

方式一：手动创建
1. 点击「创建申请」
2. 选择项目
3. 从物料下拉框选择物料（仅显示该项目未采购的物料）
4. 填写数量、期望交期、供应商等
5. 保存

方式二：导入创建
1. 点击「导入/导出」→「下载导入模板」
2. 填写模板（项目编号、物料编码、数量等）
3. 点击「导入/导出」→「导入采购申请」
4. 选择项目，上传文件

**采购申请状态**：
- 草稿
- 待审批
- 已批准
- 已驳回
- 已转订单
- 已取消

**操作说明**：

| 操作 | 说明 |
|------|------|
| 提交审批 | 将草稿状态的申请提交审批 |
| 批准 | 审批通过（需要审批权限） |
| 驳回 | 审批驳回（需要审批权限） |
| 撤回 | 撤回已提交的申请 |
| 转为订单 | 将批准的申请转为采购订单 |

### 7.3 比价分析

**路径**：ERP采购管理 → 比价分析

对比不同供应商的报价。

### 7.4 采购订单

**路径**：ERP采购管理 → 采购订单

管理正式采购订单。

**采购订单状态**：
- 草稿
- 已确认（确认后BOM状态变为"已下单"）
- 执行中
- 已收货
- 已完成
- 已取消

**确认订单**：确认后将通知供应商，并更新BOM采购状态

### 7.5 到货质检

**路径**：ERP采购管理 → 到货质检

管理采购物料的到货验收。

**收货流程**：
1. 选择采购订单
2. 录入到货数量
3. 质量检验
4. 确认入库

### 7.6 外协加工

**路径**：ERP采购管理 → 外协加工

管理外发加工的订单。

### 7.7 采购预算

**路径**：ERP采购管理 → 采购预算

制定和控制采购预算。

### 7.8 应付账款

**路径**：ERP采购管理 → 应付账款

管理供应商应付款项。

## 16. 常见问题

### Q1: 忘记密码怎么办？

联系系统管理员重置密码。

### Q2: 如何批量导入数据？

1. 进入相应模块（如客户管理、销售订单）
2. 点击「导入」或「导入/导出」按钮
3. 下载导入模板
4. 按模板格式填写数据
5. 上传文件完成导入

### Q3: 导入失败怎么处理？

- 系统会显示详细的错误信息
- 根据错误提示修改数据
- 常见问题：客户名称不匹配、必填字段为空、数据格式错误

### Q4: 如何查看审批进度？

进入「OA协同办公 → 我的提交」查看已提交的审批及其状态。

### Q5: 采购申请导入后物料需求清单没更新？

- 导入后系统会自动刷新列表
- 如未更新，请手动刷新页面

### Q6: BOM采购状态不更新怎么办？

检查采购申请/订单的操作是否完成：
- 创建采购申请 → BOM状态变为"申请中"
- 批准采购申请 → BOM状态变为"已批准"
- 确认采购订单 → BOM状态变为"已下单"

### Q7: 销售订单导入时明细没关联上怎么办？

确保明细表中填写了对应的「订单号」字段，用于关联主表的订单。

### Q8: 已关联项目的销售订单还能选择吗？

不能。已关联项目的订单会从可选列表中移除。如需重新关联，需先取消或删除原项目。
