**PRD描述：** 库存与财务对账

**解决方案：**
- StockMove统一库存流水
- 财务单据自动生成
- 完整审计日志
- 双向校验机制

### 难点2：成本归集
**PRD描述：** 将每一分钱准确算到项目

**解决方案：**
- 所有成本表强制project_id
- 三维度计算（材料+人工+费用）
- Pandas精确计算
- 实时成本追踪

✅ **已完美实现**
